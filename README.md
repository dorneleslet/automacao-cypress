# 💻 Cypress + Cucumber - Testes E2E Adopet

Este projeto contém testes end-to-end (E2E) do site **Adopet**, utilizando **Cypress em conjunto com Cucumber**.

Os cenários de teste são escritos em **Gherkin**, seguindo a abordagem **BDD (Behavior Driven Development)**, e executados através da integração entre Cucumber e Cypress.

O objetivo é garantir a qualidade das funcionalidades principais da aplicação por meio de testes automatizados, legíveis e reutilizáveis.

## 🚀 Tecnologias utilizadas

- Cypress
- Cucumber (cypress-cucumber-preprocessor)
- Gherkin
- JavaScript
- Node.js


## 🧠 Abordagem

Este projeto utiliza **BDD (Behavior Driven Development)**:

- Os testes são escritos em linguagem natural (Gherkin)
- Cada cenário descreve o comportamento esperado da aplicação
- O Cucumber interpreta os cenários e executa no Cypress


## 📌 Objetivo do projeto

Este projeto foi desenvolvido com foco em:

- Automação de testes E2E
- Boas práticas de QA
- Aplicação de BDD com Gherkin
